# 地质灾害监测设备固件 (FreeRTOS)

## 这是什么

基于 STM32L151RCT6 (Cortex-M3) 的地质灾害监测设备固件。支持温度、土壤水分/盐分、倾角、三轴加速度、激光测距、电压等多种传感器。通过 4G 模块 (SIM7600CE) 经 MQTT + 私有二进制协议与云端通信。

本工程 (`shanjing_FreeRTOS`) 是将原始裸机程序 (`shanjing/`) 按 FreeRTOS 架构重写的版本。

## 核心价值

**可靠、低功耗的野外监测** — 能在偏远地区长期无人值守运行，按时采集传感器数据并通过 4G 上传，确保数据不丢失。

## 传感器列表

- 温度 (DS18B20, 1-Wire, 最多 11 路)
- 土壤水分/盐分 (频率测量, 10 个节点)
- 倾角 (USART3, Ymodem 协议)
- 三轴加速度 (ADXL345, I2C)
- 激光测距 (SK60, 可选, USART1 分时复用)
- AD/电压 (ADS1015, I2C, 电池/太阳能/外部电压)
- 孔隙水压力 (KXYL, 可选, MDP 型号)

## 硬件平台

| 项目 | 规格 |
|------|------|
| MCU | STM32L151RCT6 (Cortex-M3, 256KB Flash, 32KB RAM) |
| 4G 模块 | SIM7600CE (USART1) |
| SPI Flash | W25X16 (2MB) |
| 外部 RTC | DS1302 (GPIO bit-bang) |
| 调试串口 | USART2 (9600 baud) |

## 关键约束

- **RAM 仅 32KB**，FreeRTOS 堆仅 8KB
- **电池供电**，3.05V 截止电压
- 必须在采集、发送、休眠之间精确平衡功耗
- Flash 无磨损均衡，需注意写入寿命

## 架构

5 个 FreeRTOS 任务 + 队列/信号量通信：

| 任务 | 优先级 | 栈 (字) | 职责 |
|------|--------|---------|------|
| Sensor | 2 | 256 | 采集所有传感器 |
| Comm | 3 | 256 | GPRS/MQTT 通信 |
| Storage | 1 | 128 | SPI Flash 数据记录 |
| Monitor | 3 | 128 | ADXL345 监测/报警 |
| Power | 1 | 128 | 电池管理/休眠调度 |

## 需求

### 已验证 (已有功能)

- ✓ **传感器采集** — DS18B20、土壤水分/盐度、倾角、ADXL345、ADS1015、SK60(可选)
- ✓ **4G 通信** — SIM7600CE AT 命令驱动、TCP 连接

- ✓ **私有二进制协议** — cigemPacket 编码/解码
- ✓ **数据存储** — SPI Flash 环形缓存
- ✓ **远程升级** — MQTT OTA + Ymodem 协议
- ✓ **低功耗管理** — RTC 唤醒、外设关断、FreeRTOS tickless idle

- [ ] **REQ-VERIFY-01**: 验证 FreeRTOS 版本完整实现了裸机版本的所有功能
